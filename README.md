# Task Manager React App

A simple and user-friendly Task Manager application built using React.

## Features
- Add tasks
- Delete tasks
- Tasks persist after refresh using localStorage
- Clean and structured dashboard layout

## Tech Stack
- React.js
- JavaScript
- CSS
- Local Storage

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Learning Outcome
- React state management
- Component-based architecture
- Local storage usage
- Clean UI structuring
